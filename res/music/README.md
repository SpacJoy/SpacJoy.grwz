# Music 文件夹

这个文件夹用于存放音乐播放器的音乐文件。

## 支持的格式

-   MP3
-   WAV
-   OGG
-   M4A
-   AAC
-   FLAC

## 使用方法

1. 将音乐文件复制到这个文件夹中
2. 刷新网页，播放器会自动检测并加载音乐
3. 如果自动检测不工作，请编辑 music-player.js 中的 manualPlaylist 数组

## 示例文件名

播放器会自动检测以下常见文件名：

-   background.mp3
-   music.mp3
-   song.wav
-   audio.ogg
-   bgm.mp3
-   theme.mp3
-   ambient.mp3
-   relaxing.mp3
-   chill.mp3

将您的音乐文件重命名为这些名称之一，或编辑 music-player.js 添加自定义文件名。
